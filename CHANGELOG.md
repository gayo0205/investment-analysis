# Changelog

本檔記錄「功能與邏輯」更新。自動資料刷新由 GitHub Actions run history 與 `Update public investment report` commit 記錄，不逐筆寫進這裡，避免每天產生無意義文字噪音。

## 2026-05-23

- 新增 TWSE ETF e添富配息清單串接：台股 ETF 卡片補「近12月配息、最近配息、下次除息、配息來源」。
- 配息來源會拆成股利、利息、收益平準金、資本利得等占比；若平準金或資本利得偏高，頁面會提醒小白不要只看殖利率。
- 定期定額模擬器改為 ETF 優先使用 Yahoo Adj Close 做含息估算，較接近配息再投入後的長期結果；若沒有 Adj Close 才退回價格口徑。
- 模擬結果新增「模擬口徑」欄位，讓使用者知道目前是含息估算或市場收盤價。
- 頁尾資料來源新增「TWSE ETF e添富配息清單」，避免配息資料來源不透明。
- 新增「今日掛單總覽」：每檔標的依價格區間、風險分數、資料可信度與市場溫度，試算買入掛單價、建議投入金額、約可買股數、最高買價、賣出觀察價與失效線。
- 今日掛單價會依台股 tick size 取合法價位；買入價用低掛邏輯，賣出價僅作觀察線，不是到價自動賣。
- 移除首頁可見的版本分工字樣，改成一般使用者看得懂的「本頁、資料檢查、固定規則」。
- 重新產生 `index.html`：2026-05-23 02:20 台灣時間，37 張卡、失敗卡 0。
- 驗證公開頁未包含 API key、授權標頭、本機路徑或私有設定名稱等敏感字串。
- 上線後移除舊的 `daily.yml` 自動更新流程，避免與新的 `update-report.yml` 三時段排程重複執行。

## 2026-05-17

- 新增 `data_sources.py`，把免費公開資料源獨立出來。
- 新增 FinMind 無 token 免費資料串接：台股個股補 PE/PBR/殖利率、月營收 YoY/MoM、近四季 EPS、EPS YoY、毛利率與三大法人買賣超。
- 新增台股 ETF 官方公開資料串接：基金基本資料、SITCA 每日淨值、FSC 境內基金費用資料。
- 台股 ETF 頁面補追蹤指數、基金類型、上市日期、官方費用月、官方 NAV 日期；ETF 費用資料若月份偏舊會標示「僅供參考」，且不拿來加分。
- 修正 TWSE/MOPS 民國日期解析，避免 0050 上市日 `09200630` 被誤顯示成西元 `0920-06-30`。
- 修正 ETF 追蹤差邏輯：只有可合理對照的標的才顯示簡易追蹤差；主題 ETF、台幣海外 ETF 與指數不匹配時改顯示 N/A 與原因。
- 新增價格分割偵測與還原價格尺度，避免 0052 這類分割後 ETF 的 52 週位置、均線與報酬被分割前價格扭曲。
- 新增報告健康檢查：若抓取失敗卡太多或卡片數不足，就保留既有 `index.html`，避免網路短暫失敗覆蓋正常公開頁。
- yfinance 快取改用 `.yf_cache_runtime/`，避免舊快取資料庫鎖定導致整批下載失敗。
- 台股盤中資料優先順序改為更清楚：盤中不讓 TWSE `STOCK_DAY_ALL` 覆蓋 Yahoo 最新/延遲報價；收盤後與休市日才用 TWSE 正式盤後資料做主價。
- 修正 Yahoo 殖利率單位防呆，避免 0.86% 被誤顯示為 86.0%。
- 台股價格改為 TWSE `STOCK_DAY_ALL` 盤後公開資料優先，Yahoo Finance 作備援與基本面/歷史資料補充。
- 新增 TWSE / Yahoo 價格尺度驗證，避免資料單位或欄位差異把 52 週位置、買進區間算壞。
- 修正 YTD 年份基準，固定使用 `Asia/Taipei`，避免 GitHub Actions UTC 時區影響。
- GitHub Actions 排程改為一天三次：美股收盤後、台股盤中、台股收盤後。
- 更新 `PUBLIC_RELEASE_AUDIT.md` 與 `README.md`，記錄資料源、限制與部署原則。
