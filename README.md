# 智慧投資分析網站

**網址：** https://gayo0205.github.io/investment-analysis/

## 這個網站做什麼？
每天早上 8:00（台灣時間）自動從 Yahoo Finance 抓取股票資料，
計算技術指標（KD、RSI、MACD、布林通道、乖離率、ADX），
以純規則判斷產生分析報告，並更新網站。

**完全免費，無任何 AI API，無任何付費服務。**

## 健康分數說明（0-100）
| 分數 | 訊號 | 意義 |
|------|------|------|
| 70-100 | 🟢 可考慮買進 | 多項指標偏多 |
| 55-69  | 🔵 持有/逢低布局 | 偏多但非強訊號 |
| 40-54  | 🟡 觀望等待 | 訊號中性或混合 |
| 0-39   | 🔴 注意風險 | 多項指標偏空 |
