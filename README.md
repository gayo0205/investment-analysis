# Investment Analysis Public Dashboard

投資分析儀表板。本版本只使用公開資料與固定規則，不包含私人持股、成本、券商資訊、API key 或 AI key。

## 內容

- 台股個股：盤中用 Yahoo 最新/延遲報價，收盤後用 TWSE 盤後公開資料；並用 FinMind 免費資料補 PE/PBR、月營收、EPS、毛利率、三大法人。
- 台股 ETF：費用率、折溢價、規模、近12月配息、配息來源、成分股集中度、總報酬估算、官方基金基本資料、官方費用月、定期定額輔助。
- 美股個股 / ETF：價格、趨勢、風險、基本面或 ETF 資料，並以台灣時間顯示報價時間。
- 債券 / 商品 ETF：先以 ETF 與風險框架觀察，仍需留意利率、期貨轉倉與匯率等本頁未完整覆蓋的因素。
- 今日掛單總覽：依價格區間、風險、資料可信度與市場溫度，試算買入掛單價、投入金額、約可買股數、最高買價、賣出觀察價與失效線。

## 自動更新

GitHub Actions 會自動執行 `generate.py` 並更新 `index.html`：

- 台灣收盤後：週一到週五 15:15 台灣時間。
- 台股盤中：週一到週五 11:30 台灣時間，用來刷新盤中/延遲報價狀態。
- 美國收盤後：週二到週六清晨 06:45 台灣時間，已晚於美股夏令/冬令收盤。
- 也可以在 GitHub Actions 頁面手動執行。

## 本機產生

```bash
pip install -r requirements.txt
python generate.py
```

## 資料與限制

- 台股價格盤中使用 Yahoo Finance，收盤後/休市日使用 TWSE 盤後公開資料優先；台股個股基本面與三大法人使用 FinMind 無 token 免費資料輔助。
- 台股 ETF 額外使用政府/公會公開資料補基金基本資料、基金費用與每日淨值，並用 TWSE ETF e添富配息清單補近12月配息、除息日與配息來源；若官方費用資料月份偏舊，頁面會標示「僅供參考」。
- 定期定額模擬器 ETF 優先用 Yahoo Adj Close 做含息估算，較接近配息再投入，但仍不是正式稅後總報酬。
- 本專案不需要、也不應放 FinMind token；若未來改用 token，只能放 GitHub Secrets 或本機環境變數。
- 報價可能是即時、延遲或收盤資料，請以頁面顯示的報價時間與資料來源為準。
- ETF 若有分割造成歷史價格尺度不同，頁面會用還原價格重算 52 週位置與均線，避免高低點誤判。
- ETF 追蹤差只在可合理對照時顯示；不能可靠估算時會顯示 N/A 與原因。
- 買進/減碼區間是紀律提醒，不是保證價格，也不是正式投資建議。
- 今日掛單價是紀律價，不是預測價；賣出觀察價不是到價自動賣，還要看轉弱、配置與風險條件。
- 個人化版本才適合加入持股、成本、目標配置、現金水位與個人買賣判斷。

## 維護文件
