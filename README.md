# Investment Analysis Dashboard

這是一個靜態投資分析儀表板，用來幫使用者快速理解市場狀態、標的位階、風險提醒與可行動的觀察方向。

## 主要功能

- 今日市場重點：整理台股、亞洲主要市場、美股、半導體與波動風險，輸出市場目前偏強、偏熱、偏冷或需要防守。
- 標的分析卡：針對 ETF、個股、債券、商品與加密資產，以不同邏輯顯示趨勢、風險、價格區間與資料可信度。
- 動態價格地圖：使用均線、ATR 與布林通道輔助判斷可分批區、追高上限、停買區與風險線。
- 定期定額模擬：協助理解長期投入、最大回撤與總報酬口徑的差異。
- 新手翻譯：重要數字會附上白話解釋，避免只堆 RSI、MACD、KD 等技術指標。

## 資料來源

- Yahoo Finance：價格、成交量、部分指數與還原價格資料。
- TWSE 公開資料：台股 ETF 配息、除息與相關公開資訊。
- FinMind 免費資料：台股基本面與月營收等公開資料。

資料來源可能有延遲、缺漏或格式異動；頁面會盡量標示資料時間與可信度。

## 自動更新

GitHub Actions 會排程執行 `generate.py` 並更新 `index.html`。

目前設計為每天多次更新：

- 台股盤後：用於更新台股收盤資料。
- 美股盤後：用於更新美股與全球指數資料。
- 盤中檢查：用於補資料與降低單一資料源延遲造成的落差。

## 本機執行

```bash
pip install -r requirements.txt
python generate.py
```
