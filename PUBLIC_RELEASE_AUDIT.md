# Public Release Audit

這份清單用來記錄目前的資料邏輯、限制與下一步。本專案不能放私人持股、成本、券商設定、API key 或 AI key。

## 目前已完成

- `generate.py`：產生靜態 `index.html`，使用公開資料與固定規則。
- `index.html`：展示頁，包含市場溫度、定期定額模擬、今天想買試算、題材觀察、計算方式、資料檢查。
- `.github/workflows/update-report.yml`：GitHub Actions 自動更新，每天最多三次。
- `requirements.txt`：Actions 安裝必要套件。
- `.gitignore`：排除 `.yf_cache/`、`__pycache__/`、`.env`、log。

## 自動更新時段

- 美股收盤後：週二到週六清晨 06:45 台灣時間。
- 台股盤中：週一到週五 11:30 台灣時間。
- 台股收盤後：週一到週五 15:15 台灣時間。

## 目前資料源

- 台股價格：盤中使用 Yahoo Finance 最新/延遲報價；收盤後、休市日、開盤前使用 TWSE 盤後公開資料優先，Yahoo Finance 備援。
- 美股、海外 ETF、指數與部分基本面：主要由 Yahoo Finance 取得。
- 台股報價時間：頁面以台灣時間為主，必要時括號顯示原市場時間。
- 台股個股基本面與籌碼：FinMind 無 token 免費資料補 PE/PBR/殖利率、月營收、EPS、毛利率與三大法人；Yahoo Finance 補 Forward PE、ROE 等欄位。
- 台股 ETF：Yahoo Finance 補費用率、折溢價、規模、配息、成分股與歷史價格；政府/公會公開資料補基金基本資料、追蹤指數、基金類型、上市日期、基金費用月與每日淨值；TWSE ETF e添富配息清單補近12月配息、除息日與配息來源組成。

## 時間邏輯

- 報告產生時間固定使用 `Asia/Taipei`。
- Yahoo 報價時間使用 `regularMarketTime` 與 `exchangeTimezoneName`，頁面主顯示台灣時間。
- TWSE `STOCK_DAY_ALL` 是盤後公開資料，不是即時價；程式用資料日 `14:30 Asia/Taipei` 作為盤後資料時間標記。
- 若 ETF 曾分割或價格尺度跳動，程式會用還原價格尺度重算 52 週位置與均線，避免拿分割前價格和現在價格比較。
- 自動更新歷史由 GitHub Actions run history 與更新 commit 保留；功能邏輯更新寫入 `CHANGELOG.md`。

## 需要升級的資料源

- 台股 OHLC / 成交量：已先接 TWSE `STOCK_DAY_ALL` 盤後公開資料；仍需再補更長歷史資料來源。
- 台股即時/盤中：TWSE 正式即時資料有授權與費用問題；本頁只適合標示延遲或用免費來源輔助。
- 台股基本面：FinMind 已補月營收、EPS、毛利率、PE/PBR/殖利率；ROE、現金流仍待更正式來源。
- 台股 ETF：已補一部分官方公開資料與 TWSE ETF e添富配息資料；仍需補完整成分股、正式折溢價、正式追蹤誤差。
- 籌碼：三大法人已補；融資融券、外資持股待補。
- 債券 / 商品：TLT/IEF 應補利率與殖利率曲線；USO/DBC 應補期貨轉倉成本。

## 仍需討論

- FinMind 目前使用無 token 模式。若未來使用 token，必須只放 GitHub Secrets 或本機環境變數，不能寫進 repo。
- 台股盤中更新要不要提高頻率。目前 11:30 只刷新一次，避免免費資料被限流。
- 是否要繼續拆檔：目前已先新增 `data_sources.py`，後續可再拆 `scoring.py`、`render.py`。
- 後續個人化資料格式：持股、成本、目標配置、現金水位、券商手續費、投資目的。

## 目前已知限制

- 定期定額模擬器已優先用 Yahoo Adj Close 做含息估算，但仍不是正式稅後總報酬；未完整計入匯率、稅務、券商差異與ETF內扣成本。
- TWSE ETF e添富配息組成占比屬公告預估，不能當成未來配息保證。
- 今日掛單總覽是規則化試算：買入掛單價用價格區間與 tick size，賣出觀察價要搭配轉弱、超過配置或 ETF 異常，不是到價自動賣。
- 買進/減碼區間是規則化紀律，不是預測最低點或最高點。
- Yahoo Finance 免費資料可能延遲、欄位缺漏或偶爾與交易所正式資料不同。
- 官方基金費用資料可能不是最新月份；頁面會標示資料偏舊，且不把偏舊費用拿去加分。
- SITCA 每日淨值不一定能用 ETF 交易代號完整對上；對不上時顯示 `N/A` 與待補。
- ETF 追蹤差不是每檔都能可靠估算；主題 ETF、台幣海外 ETF 或基準指數不匹配時，會顯示未估原因而不是硬給數字。
- 若資料抓取失敗過多，產生器會保留既有 `index.html`，避免短暫網路問題把公開頁覆蓋成失敗版。
- 本頁不會真正判斷「你本人該買/賣幾股」，這要留到後續個人化版本。
